Inspetor de elementos

Voltar ao inicio

O inspetor de elementos é uma das grandas ferramentes indispensaveis para todo tipo de programador, nos permitindo entender como os elementos estão se comportando dentro de uma página web, assim possibilitando um maior controle dos efeitos das várias tags em uma página.

Estrutura básica do HTML

Voltar ao inicio

A criação de páginas web com HTML segue um padrão bastante restrito, com uma estrutura constituida por 3 tags básicas, a primeira sendo a tag "html", que abre no inicio do documento e fecha na última linha do documento, assim sendo a base do documento html

Logo após a tag "html", temos a tag "head", que define todos os elementos que deveram ser carregados antes da pagina ser aberta para visualização,estes elementos são chamados de metadados

Por fim a tag "body", sendo o corpo da página html, onde seram exibidos todos os conteúdos visuais para o usuário, como imagens, textos, videos e diversos outros tipos de mídias.

Tags Básicas

Voltar ao inicio

O html possui milhares de tags, mas algumas são bem mais comuns que outras, como exemplo de tags muito utilizadas temos as tags de marcação de texto, "strong" que tem como função deixar uma palavra, frase ou até uma grande texto em negrito, a tag "i" que tem como função transformar tudo que estiver dentro dela em italico, estas duas tags podem ser combinadas, para criar bons efeitos para um texto.

Outra tag bastante utilizada é a "input", que cria uma caixa de inserção de dados, tanto textos como números.

Tags de textos

Voltar ao inicio

Sendo uma linguagem de marcação de texto, O HTML dispõe de diversas tags para criação e edição de textos dos mais diversos tipos, mas temos como principais a tag "p" que é utilizada para a criação e separação de paragrafos, e as tags de "h1" até "h6", que são utilizados para a criação de títulos, quanto menor o número acompanhado do "h", maior sera o tamanho das letras empregadas a palavra ou frase.

Listas ordenadas e não ordenadas

Voltar ao inicio

Com o HTML podemos criar alguns tipos de listas, com itens ordenados ou não ordenados, para isso devemos iniciar a estrutura com as tags "ul" para listas não ordenadas ou "ol" para listas ordenadas, após isso devemos inserir os itens desta lista, que devem estar na tag "li" assim adicionando os itens a nossa lista.
